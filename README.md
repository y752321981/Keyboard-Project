，其实就是按哪个建对应按键就闪一下，程序运行时会隐藏任务栏，强制置顶，关闭之后会还原，支持自定义键盘样式，~~（但是除了图中的按键，我都没去做获取输入的处理，所有只能减，不能加）~~。

点击右上角设置样式按钮后就可以进行更改按键的位置，增添按键，读档存档一系列操作。
